Данная библиотекка позволяет считывать любой буфер и по мере продвижения интерпретировать результат, а затем регистрировать методы-обработчики для каждого считанного объекта.
Позволяет с помощью файла конфигурации составленного по одределенным правилам, указанным в документации, строить pipeline
